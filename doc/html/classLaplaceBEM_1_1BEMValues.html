Detailed Description

template<int dim, int spacedim, typename RangeNumberType>
class LaplaceBEM::BEMValues< dim, spacedim, RangeNumberType >

Values related to a pair of cells (panels) used in Galerkin BEM, which can be considered as a counterpart of the FEValues for FEM in deal.ii.

Note
The values encapsulated in BEMValues are shape functions value and their gradient values at each quadrature point, as well as Sauter quadrature rules for different cell neighboring types. These values will be precalculated for improving the performance.
